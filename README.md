#  Fast Food Nutrition Analytics (Power BI + SQL)

##  Overview

This project analyzes the **nutritional composition of fast food items** using Power BI and SQL.
The goal is to uncover unhealthy consumption patterns, compare restaurant offerings, and provide data-driven insights for better dietary decisions.

---

##  Problem Statement

How can we identify unhealthy food patterns across fast food restaurants and help users make healthier choices using data?

---

##  Dataset Details

* Contains **500+ fast food items**
* Includes multiple restaurant brands
* Key attributes:

  * Calories
  * Protein
  * Fat
  * Sugar
  * Fiber
  * Restaurant

---

##  Data Cleaning & Transformation

Performed using **Power Query Editor**:

* Removed columns with high missing values
* Handled null values
* Corrected data types
* Standardized column names
* Removed irrelevant columns

---

##  Tools & Technologies

* **Power BI** → Dashboard & Data Visualization
* **Power Query** → Data Cleaning
* **SQL** → Data Analysis & Business Queries

---

##  Key KPIs

* 🔹 Average Calories (530)
* 🔹 Average Protein (28g)
* 🔹 Average Fat (27g)
* 🔹 Total Items (512)
* 🔹 Healthy vs Unhealthy Distribution

---

##  Dashboard Features

### 🔍 Interactive Filters

* Restaurant filter
* Health status filter (Healthy / Unhealthy)
* Calories range slider

###  Visualizations

* Scatter Plot → Calories vs Protein
* Bar Chart → Average Calories by Restaurant
* Bar Chart → Top 5 High-Calorie Items
* Bar Chart → Top 5 High-Sugar Items
* Donut Chart → Healthy vs Unhealthy Distribution
* Table → Detailed nutritional data

---

##  Key Insights

* High-calorie items are not always high in protein
* Some items contain extremely high sugar levels
* Significant variation exists across restaurants
* A small number of items contribute heavily to unhealthy consumption
* Certain restaurants consistently offer higher-calorie menus

---

##  Business Use Case

This dashboard can help:

* 🍽️ Customers make healthier food choices
* 🏢 Restaurants optimize menu offerings
* 📊 Analysts understand food consumption patterns

---

## 🧾 SQL Analysis

SQL queries were used to:

* Identify high-calorie and high-sugar items
* Compare restaurants based on nutritional metrics
* Categorize items into healthy vs unhealthy
* Calculate calorie-to-protein efficiency
